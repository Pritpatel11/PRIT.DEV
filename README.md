# 🚀 PRIT_MODERN.DEV — Personal Developer Portfolio

A futuristic, animation-rich developer portfolio built with modern web technologies. Features cyberpunk aesthetics, GSAP scroll animations, creative custom cursor, and a terminal-style contact form with Google Sheets integration.

## ⚡ Live Demo
🔗 [https://pritpatel.dev](https://pritdev.netlify.app)

## ⚙️ Tech Stack
- **Frontend:** React.js, Vite
- **Animations:** GSAP, Framer Motion
- **Styling:** Tailwind CSS
- **SEO:** React Helmet Async, JSON-LD Structured Data
- **Contact:** Google Apps Script (Sheets + Email)

## 🎨 Features
- 🖱️ Creative custom cursor with velocity-based stretching & particle trail
- 📄 Animated project case study pages with GSAP ScrollTrigger
- ⭐ Client testimonials carousel with star ratings
- 🧠 Neural Sources section crediting learning mentors
- 📬 Terminal-style contact form → Google Sheets + Email
- 📱 Fully responsive with pro-level mobile menu
- 🔍 100% SEO optimized (Meta tags, OG, Twitter Cards, Sitemap, robots.txt)
- 🎨 Theme accent color switcher

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/pritpatel11/portfolio.git

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
